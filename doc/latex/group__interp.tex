Compute the weights for direct differentiation at a point, based on values at the base points of a system of orthogonal polynomials, so that $\partial^{d_{1}+d_{2}+\dots}f(\mathbf{x})/\partial x_{1}^{d_{1}}x_{2}^{d_{2}}\dots\approx\sum v_{i}f_{i}$. Remember to use \+::mop\+\_\+polynomial\+\_\+index to map base points to real points.
